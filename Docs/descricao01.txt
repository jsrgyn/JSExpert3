Sejam muito bem vindos a #SemanaJSExpert​ 3.0

Nessa aula você vai construir a estrutura inicial do nosso programa de linha de comando além de configurar toda a interface do programa usando o pacote Blessed e o design pattern Builder.

Vai aprender a trabalhar com o módulo nativo do Node.js para emissão de eventos customizados e definir quais eventos sua interface deverá respeitar.

Você também vai trabalhar com emissores de evento para estabilizar a comunicação entre componentes e regras de negócio.
